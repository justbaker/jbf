# jbf

A [brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter in JS/Es6

## Usage
Brainfuck hello world program:

```
++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
```

Running interpreter and hello world program:

```
npm start

Justin's Brainfuck Interpreter v.0.1.0
======================================
brainfuck $ ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.
Hello World!
```
